Profil Singkat

Saya adalah pengembang web yang berfokus pada ekosistem e-commerce dan aplikasi interaktif. Dengan latar belakang akademis di bidang Software Quality Assurance (SQA) dan Cybersecurity, saya tidak hanya membangun antarmuka yang menarik secara visual, tetapi juga memastikan kode yang saya tulis aman, tahan banting, dan telah melewati skenario pengujian yang ketat. Saya memiliki pengalaman dalam membangun fitur dinamis menggunakan Shopify Liquid dan React, serta mengelola lingkungan server berbasis Linux.

Shopify (Liquid, Schema, AJAX API), React, Flutter, JavaScript, HTML/CSS.

Software Quality Assurance

Skenario Pengujian (White Box / Black Box), Equivalence Partitioning, Code Coverage Metrics.

Proyek Unggulan

1. Custom Shopify Theme: Dynamic Storefront & AJAX Cart Drawer

Teknologi: Shopify Liquid, JavaScript (Fetch API), CSS3, JSON Schema.

Membangun antarmuka toko online interaktif yang responsif dan mudah dikelola oleh admin.

  • Merancang Custom Hero Section dan Product Grid dinamis menggunakan Liquid looping ({% for %}) dan konfigurasi Schema.
  • Mengembangkan Cart Drawer interaktif menggunakan manipulasi DOM asinkron.
  • Penerapan SQA: Mengimplementasikan pengujian fallback ({% if product != blank %}) untuk mencegah halaman crash saat data kosong.

2. Cinema-kampus-UAS (Web Application)

Teknologi: React, Git, Vercel.

Aplikasi pustaka film modern untuk memenuhi kebutuhan manajemen katalog hiburan digital.

Deskripsi: Membangun antarmuka aplikasi dengan struktur komponen React yang dapat digunakan ulang (reusable). Proyek ini dikelola menggunakan Git dan di-deploy secara live menggunakan Vercel untuk memastikan aksesibilitas publik yang cepat dan andal.

3. Web Vulnerability Testing & Mitigation (Laboratorium XSS)

Fokus: Cybersecurity & Secure Coding

Analisis keamanan aplikasi web terhadap serangan injeksi skrip.

Deskripsi: Melakukan identifikasi, eksploitasi terkontrol, dan mitigasi pada berbagai skenario serangan Cross-Site Scripting (Reflected XSS, filter bypassing, tag filtering). Memastikan penerapan praktik coding yang aman (secure coding) saat menangani input pengguna.
